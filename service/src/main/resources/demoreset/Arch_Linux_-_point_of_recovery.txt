Content: 

<p>If&nbsp;for&nbsp;some&nbsp;reason&nbsp;(like&nbsp;when&nbsp;there&nbsp;was&nbsp;a&nbsp;system&nbsp;upgrade&nbsp;and&nbsp;you&nbsp;can&#39;t&nbsp;use&nbsp;GUI)&nbsp;you&nbsp;need&nbsp;to&nbsp;go&nbsp;back&nbsp;in&nbsp;time&nbsp;to&nbsp;a&nbsp;specific&nbsp;version&nbsp;of&nbsp;your&nbsp;system&nbsp;you&nbsp;can&nbsp;use&nbsp;pacman&nbsp;and&nbsp;essentially&nbsp;do&nbsp;a&nbsp;&quot;recovery&quot;.&nbsp;Go&nbsp;to&nbsp;your&nbsp;pacman&nbsp;directory&nbsp;and&nbsp;find&nbsp;the&nbsp;&quot;mirrorlist&quot;&nbsp;file.</p><p></p><p>An&nbsp;example&nbsp;of&nbsp;the&nbsp;path:</p><pre data-language="plain">
etc/pacman.d/mirrorlist
</pre><p></p><p>An&nbsp;example&nbsp;of&nbsp;the&nbsp;file&#39;s&nbsp;contents:</p><p></p><pre data-language="plain">
# Server list generated by rankmirrors on 2024-04-28
################################################################################
################# Arch Linux mirrorlist generated by Reflector #################
################################################################################
# With:       reflector @/etc/xdg/reflector/reflector.conf
# When:       2024-04-28 10:06:10 UTC
# From:       https://archlinux.org/mirrors/status/json/
# Retrieved:  2024-04-28 10:05:20 UTC
# Last Check: 2024-04-28 09:34:28 UTC

Server = https://mirror.osbeck.com/archlinux/$repo/os/$arch
Server = https://mirror.moson.org/arch/$repo/os/$arch
Server = https://mirror.sunred.org/archlinux/$repo/os/$arch
Server = https://mirror.f4st.host/archlinux/$repo/os/$arch
Server = https://archlinux.thaller.ws/$repo/os/$arch
Server = https://mirror.cyberbits.eu/archlinux/$repo/os/$arch
</pre><p></p><p>What&nbsp;do&nbsp;you&nbsp;want&nbsp;to&nbsp;do&nbsp;is&nbsp;to&nbsp;add&nbsp;a&nbsp;single&nbsp;line&nbsp;like&nbsp;this:</p><p></p><pre data-language="plain">
Server = https://archive.archlinux.org/repos/2025/04/25/$repo/os/$arch
</pre><p></p><p>and&nbsp;also&nbsp;you&nbsp;need&nbsp;to&nbsp;delete&nbsp;or&nbsp;comment&nbsp;out&nbsp;the&nbsp;remaining&nbsp;endpoints:</p><p></p><pre data-language="plain">
  # Server list generated by rankmirrors on 2024-04-28
################################################################################
################# Arch Linux mirrorlist generated by Reflector #################
################################################################################
# With:       reflector @/etc/xdg/reflector/reflector.conf
# When:       2024-04-28 10:06:10 UTC
# From:       https://archlinux.org/mirrors/status/json/
# Retrieved:  2024-04-28 10:05:20 UTC
# Last Check: 2024-04-28 09:34:28 UTC

Server = https://archive.archlinux.org/repos/2025/04/25/$repo/os/$arch
#Server = https://mirror.osbeck.com/archlinux/$repo/os/$arch
#Server = https://mirror.moson.org/arch/$repo/os/$arch
#Server = https://mirror.sunred.org/archlinux/$repo/os/$arch
#Server = https://mirror.f4st.host/archlinux/$repo/os/$arch
#Server = https://archlinux.thaller.ws/$repo/os/$arch
#Server = https://mirror.cyberbits.eu/archlinux/$repo/os/$arch
</pre><p></p><p>Then&nbsp;you&nbsp;can&nbsp;save&nbsp;the&nbsp;file&nbsp;(most&nbsp;likely&nbsp;you&nbsp;will&nbsp;need&nbsp;to&nbsp;the&nbsp;whole&nbsp;thing&nbsp;as&nbsp;a&nbsp;root)&nbsp;and&nbsp;just&nbsp;upgrade&nbsp;the&nbsp;system.&nbsp;If&nbsp;that&nbsp;didn&#39;t&nbsp;work&nbsp;you&nbsp;probably&nbsp;just&nbsp;need&nbsp;to&nbsp;insert&nbsp;an&nbsp;earlier&nbsp;date.</p>
